# 📌 About The Project

**Foundational AI Lab** is a structured repository dedicated to understanding Artificial Intelligence from first principles through:

- Mathematical intuition
- Scratch implementations
- Real-world experimentation
- Research-oriented exploration
- Practical deep learning workflows

This repository is built as a long-term AI engineering + research foundation covering:

✅ Machine Learning  
✅ Deep Learning  
✅ Optimization Techniques  
✅ Neural Networks  
✅ CNN Architectures  
✅ PyTorch Fundamentals

✅ Experimentation & EDA

# 🚀 Setup Instructions

## 1️⃣ Clone Repository

```bash
git clone https://github.com/RajeshPhulwaria006/AI-journey-foundation.git
```

---

## 2️⃣ Move Into Project Directory

```bash
cd AI-journey-foundation
```

---

## 3️⃣ Create Virtual Environment

### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

---

## 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```
